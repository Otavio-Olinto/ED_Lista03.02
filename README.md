# ED_Lista03.02
## Algarismos Presentes: Recursiva

Este desafio consiste em criar uma aplicação, em Java, que receba dois números naturais e apresente quantas vezes o algarismo de um aparece no outro.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. O primeiro número deve ser de 10 a 999.999;
3. O segundo número deve ser de 0 a 9;
4. A operação deve ser realizada por uma função recursiva;
5. Declarar, via comentário, a condição de parada da função.
```
